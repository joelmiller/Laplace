1.8. Using the Laplace Transform to solve Differential Equations#

1.8.1. Intended Learning Outcomes#

By the end of this section, you should be able to:

  • Express the Laplace transforms of derivatives:

\[\begin{align*} \mathcal{L}[f'(t)] &= s\tilde{f}(s) - f(0)\\ \mathcal{L}[f''(t)] &= s^2\tilde{f}(s) - sf(0) - f'(0)\end{align*}\]

  • Apply these formulas to convert a linear, constant-coefficient first or second order ODEs in \(t\) into an algebraic equation in \(s\).

  • Solve the resulting equation for \(\tilde{f}(s)\) and then invert it.

  • Handle cases that involve discontinuous forcing functions.

1.8.2. Introduction#

We now begin to apply the Laplace Transform to solving constant coefficient linear differential equations. We focus on differential equations of the form

\[f'(t) + af(t) = g(t)\]

and

\[f''(t) + af'(t) + bf(t) = g(t)\]

The steps in this process are:

  • Take the Laplace Transform of both sides of the differential equation (using the initial conditions)

  • Do some Algebra to solve for \(\tilde{f}(s)\).

  • Invert the Laplace Transform of \(\tilde{f}(s)\) to find \(f(t)\).

1.8.3. Transforms of \(f'(t)\) and \(f''(t)\)#

To take the Laplace Transform of our differential equations, we will need to know the transforms of \(f'(t)\) and \(f''(t)\).

Theorem 1.5 (Transforms of derivatives)

Given a function \(f(t)\) whose Laplace Transform is \(\tilde{f}(s)\), the transform of \(f'(t)\) and \(f''(t)\) are

\[\begin{align*} \mathcal{L}[f'(t)](s) &= s\tilde{f}(s) - f(0)\\ \mathcal{L}[f''(t)](s) &= s^2 \tilde{f}(s) - sf(0) - f'(0) \end{align*}\]

Proof. To demonstrate this for \(f'(t)\), we turn to the definition of the Laplace Transform:

\[\begin{align*} \mathcal{L}[f'(t)](s) &= \int_0^\infty f'(t) e^{-st} \, dt\\ &= \left. f(t)e^{-st} \right|_0^\infty - \int_0^\infty f(t)(-s)e^{-st} \, dt\\ &= 0 - f(0) + s \int_0^\infty f(t) e^{-st} \, dt\\ &= -f(0) + s \tilde{f}(s) \end{align*}\]

To demonstrate this for \(f''(t)\), we rewrite the rule for \(f'(t)\) as

\[ \mathcal{L}\left[\frac{d}{dt}f'(t)\right](s) = -f(0) + s\mathcal{L}[f(t)](s) \]

Then applying this rule twice gives

\[\begin{align*} \mathcal{L}[f''(t)](s) &= \mathcal{L}[\frac{d}{dt} f'(t)](s)\\ &= -f'(0) + s \mathcal{L}[f'(t)](s)\\ &= -f'(0) + s(s\tilde{f}(s) - f(0))\\ &= s^2\tilde{f}(s) - sf(0) - f'(0) \end{align*}\]

So the transforms of \(f'(t)\) and \(f''(t)\) can be written in terms of the transform of \(f(t)\) and the initial conditions. This is the key detail that leads to the Laplace Transform of a differential equation for \(f(t)\) becoming an Algebraic equation for \(\tilde{f}(s)\).

1.8.4. Solving Differential Equations: Examples#

Example 1.30 (\(f''(t) + 4f'(t) + 3f(t) = -6e^{-3t}\))

To find \(f(t)\) if \(f''(t) + 4f'(t) + 3f(t) = -6e^{-3t}\) with \(f(0) = 1\) and \(f'(0)=-2\) we take the Transform of both sides:

\[\begin{align*} f''(t) + 4f'(t) + 3f(t) &= -6e^{-3t}\\ \Rightarrow \quad \mathcal{L}[f''(t) + 4f'(t) + 3f(t)](s) &= \mathcal{L}\left[-6e^{-3t}\right](s)\\ s^2\tilde{f}(s) - sf(0) - f'(0) + 4(s\tilde{f}(s) - f(0)) + 3 \tilde{f}(s) &= \frac{-6}{s+3}\\ (s^2+4s+3)\tilde{f}(s) - s + 2 -4 &= \frac{-6}{s+3}\\ \Rightarrow \quad \tilde{f}(s) &= \frac{-6}{(s+3)(s^2+4s+3)} + \frac{s+2}{s^2+4s+3}\\ &= \frac{-6}{(s+3)^2(s+1)} + \frac{s+2}{(s+3)(s+1)} \end{align*}\]

We have solved for \(\tilde{f}(s)\). To find \(f(t)\), we need to invert the transform. We could approach each fraction separately or we could do a single partial fractions expansion. Typically if you can’t immediately see what the expansion of a given fraction looks like, it is easiest to do all the terms together in a single expansion. The common denominator is \((s+3)^2(s+1)\). So we have

\[ \frac{-6}{(s+3)^2(s+1)} + \frac{s+2}{(s+3)(s+1)} = \frac{A}{(s+3)^2} + \frac{B}{s+3} + \frac{C}{s+1} \]

We multiply by the common denominator:

\[ -6 + (s+2)(s+3) = A(s+1) + B(s+3)(s+1) + C(s+3)^2 \]

Notice that we do not expand the product on the left hand side. Multiplying it out would make the next calculations more difficult (and give an opportunity for errors). Using \(s=-3\):

\[ -6 = -2A \Rightarrow A=3 \]

Using \(s=-1\):

\[ -6 +(-2)(-1) = 4C \Rightarrow C=-1 \]

Finally another convenient value is \(s=0\):

\[ 0 = A + 3B + 9C = 3-9+3B \Rightarrow B=2 \]

Thus

\[ \tilde{f}(s) = \frac{3}{(s+3)^2} + \frac{2}{s+3} + \frac{-1}{s+1}\]

and

\[\begin{align*} f(t) &= \mathcal{L}^{-1}[\tilde{f}(s)](t)\\ &= \mathcal{L}^{-1}\left[\frac{3}{(s+3)^2} + \frac{2}{s+3} + \frac{-1}{s+1}\right](t)\\ &= 3\mathcal{L}^{-1}\left[\frac{1}{(s+3)^2}\right](t) + 2\mathcal{L}^{-1}\left[\frac{1}{s+3}\right](t) - \mathcal{L}^{-1}\left[\frac{1}{s+1}\right](t)\\ &= 3e^{-3t} \mathcal{L}^{-1}\left[\frac{1}{s^2}\right](t) + 2 e^{-3t} - e^{-t}\\ &= 3te^{-3t}+ 2e^{-3t} - e^{-t} \end{align*}\]

It is useful to see an example that involves a discontinuity:

Example 1.31 (\(f'(t) + 2f(t) = t H_4(t)\))

To solve for \(f(t)\) when \(f'(t) + 2f(t) = t H_4(t)\) with \(f(0) = 3\) we follow the same steps, taking the Laplace Transform of both sides.

We have

\[\begin{align*} \mathcal{L}[f'(t) + 2f(t)](s) &= \mathcal{L}[t H_4(t)](s)\\ s\tilde{f}(s) - 3 + 2\tilde{f}(s) &= \mathcal{L}[t H_4(t)](s) \end{align*}\]

To do the transform of the discontinuous term, we will look at two equivalent approaches. You can use whichever one is more convenient, but it would be good to have practice with both:

  • From the Transform table, we have

    \[ \mathcal{L}[g(t)H_a(t)](s) = e^{-as} \mathcal{L}[g(t+a)](s) \]

    Using \(g(t) = t\), this says

    \[ \mathcal{L}[tH_4(t)](s) = e^{-4s}\mathcal{L}[t+4](s) \]

  • Alternately, from the Transform table, we have

    \[ \mathcal{L}[g(t-a)H_a](s) = e^{-as} \mathcal{L}[g(t)](s) \]

    To use this rule, we replace \(t\) by \((t-a)+a\). In this case, replacing \(t\) by \((t-4)+4\), we can write \(t=g(t-4)\) where \(g(x) = x+4\). So using \(g(t)=t+4\), this says

    \[ \mathcal{L}[tH_4(t)](s) = e^{-4s}\mathcal{L}[t+4](s) \]

Either way we end up with the same thing, \(e^{-4s} \mathcal{L}[t+4](s) = e^{-4s} \left(\frac{1}{s^2} + \frac{4}{s}\right)\). Thus we have

\[\begin{align*} (s+2) \tilde{f}(s) -3 &= e^{-4s} \left(\frac{1}{s^2} + \frac{4}{s}\right)\\ \tilde{f}(s) &= \frac{3}{s+2} + e^{-4s} \frac{1+4s}{s^2(s+2)} \end{align*}\]

We have \(\tilde{f}(s)\) so we are ready to find \(f(t)\) by inverting the Laplace Transform. When we invert the transform, we need to handle the part with \(e^{-4s}\) and the part without \(e^{-4s}\) separately.

\[\begin{align*} f(t) &= \mathcal{L}^{-1}[\tilde{f}(s)](t)\\ &= \mathcal{L}^{-1}\left[ \frac{3}{s+2} + e^{-4s} \frac{1+4s}{s^2(s+2)}\right](t)\\ &= 3e^{-2t} + \mathcal{L}^{-1} \left[ e^{-4s}\frac{1+4s}{s^2(s+2)}\right](t)\\ &= 3e^{-2t} + H_4(t) \mathcal{L}^{-1} \left[ \frac{1+4s}{s^2(s+2)}\right](t-4) \end{align*}\]

We will use a partial fractions expansion

\[\frac{1+4s}{s^2(s+2)} = \frac{A}{s^2} + \frac{B}{s} + \frac{C}{s+2} \]

Multiplying through by the denominator, we get

\[ 1+4s = A(s+2) + B(s)(s+2) + Cs^2 \]

Taking \(s=-2\), we have \(4C = -7\) and \(C = -7/4\). Taking \(s=0\) gives \(2A=1\), so \(A=1/2\). It is convenient to look at the coefficient of \(s^2\) to conclude \(B+C=0\) so \(B=-C = 7/4\).

So we finally have

\[\begin{align*} f(t) &= 3e^{-2t} + H_4(t) \mathcal{L}^{-1} \left[ \frac{1+4s}{s^2(s+2)}\right](t-4)\\ &= 3e^{-2t} + H_4(t) \mathcal{L}^{-1} \left[\frac{1/2}{s^2} + \frac{7/4}{s} - \frac{7/4}{s+2} \right](t-4)\\ &= 3e^{-2t} + H_4(t) \left( \frac{1}{2}(t-4) + \frac{7}{4} - \frac{7}{4}e^{-2(t-4)}\right) \end{align*}\]

Notice that even though the function that was on the right hand side of the differential equation is discontinuous, the resulting function is continuous (you can check that the term multiplied by \(H_4(t)\) that turns on at \(t=4\) is \(0\) right at \(t=4\)). So even though the forcing changes discontinuously, as long as it is finite, the response \(f(t)\) changes as a continuous variable.
