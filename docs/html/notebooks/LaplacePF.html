1.6. Inverting the Laplace Transform with Partial Fractions — quadratic denominators#

When we later apply the Laplace Transform to a differential equation for \(f(t)\), we will frequently find that \(\tilde{f}(s)\) is equal to \(p(s)/q(s)\) where \(p\) and \(q\) are both polynomials (with \(p\) having lower order than \(q\)).

Sometimes we end up with something like \(\tilde{f}(s) = \frac{2}{s-1}\), which is straightforward to solve from the transform table because \(\frac{1}{s-a}\) is in it. However, often we end up with something more complicated. For example, we might have \(\tilde{f}(s) = \frac{s+3}{s^2+3s+2}\). We do not have something like this in our table.

We will use Partial Fractions to convert \(\tilde{f}(s)\) into a sum of functions that do appear in our table. For example, we will learn to rewrite \(\tilde{f}(s) = \frac{s+3}{s^2+3s+2}\) as \(\tilde{f}(s) = \frac{2}{s+1} -\frac{1}{s+2}\). We will then use linearity to conclude \(f(t) = 2\mathcal{L}^{-1}[1/(s+1)](t) - \mathcal{L}^{-1}[1/(s+2)](t)\), which reduces this to things that are in the table.

1.6.1. Rational Functions#

Definition 1.3 (Rational Function)

A rational function is a function that can be written in the form

\[ f(x) = \frac{p(x)}{q(x)} \]

where both \(p(x)\) and \(q(x)\) are polynomials.

We can always assume that the leading coefficient of \(q(x)\) is \(1\), because if it is not, we can divide both \(p(x)\) and \(q(x)\) by that coefficient to make it \(1\).

We will often need to find \(f(t)\) when \(\tilde{f}(s) = \frac{p(s)}{q(s)}\) where \(p(s)\) and \(q(s)\) are both polynomials and the highest degree of \(p(s)\) is less than the highest degree of \(q(s)\).

1.6.2. Classifying the factorizations of \(q(s)\)#

Consider

\[ \tilde{f}(s) = \frac{p(s)}{q(s)} \]

where the highest degree of \(p(s)\) is \(0\) or \(1\) and the highest degree of \(q(s)\) is \(2\) and the leading coefficient of \(q(s)\) is \(1\).

Our goal will be to find \(f(t)\). Typically under these assumptions, \(\tilde{f}(s)\) will not appear in our Laplace Transform Table. We will find a way to write \(\tilde{f}(s)\) as the sum of two functions that do appear in the table. Our first step is to find two candidate functions to make up the sum and the next step is to determine the appropriate coefficients. To find those candidate functions, we first need to rewrite \(q\) into one of these three possibilities:

  • \(q(s) = (s-a)(s-b)\) with \(a \neq b\): there are two real unequal roots to the equation \(q(s)=0\).

  • \(q(s) = (s-a)^2\): there is a single repeated real root to the equation \(q(s) = 0\).

  • \(q(s) = (s-a)^2 + \omega^2\): there are no real roots to the equation \(q(s)=0\).

We will use completing the square or direct factorization to determine which of these three cases applies. Once we have classified \(q(s)\) based on this, we will be able to set up Partial Fractions to find \(f(t) = \mathcal{L}^{-1}[p(s)/q(s)](t)\).

1.6.2.1. Completing the square#

An important step in our process will be to take a quadratic polynomial and use “completing the square” to change its structure. The goal is to first convert the quadratic formula into the form \((s+c_1)^2 + c_2\). From there we will be able to convert it to one of the three forms identified above, depending on whether \(c_2<0\), \(c_2=0\), or \(c_2>0\). Rather than trying to memorize a formula for completing the square we will learn a process, starting with an example.

Before you go further, please remind yourself of a common error: \((a+b)^2 \neq a^2 + b^2\). Instead \((a+b)^2 = a^2 + 2ab + b^2\). The \(2ab\) term plays an important role in what follows.

Example 1.18 (Completing the square when \(q\) does not factor: \(q(s) = s^2 + 6s + 13\))

We have \(q(s) = s^2 + 6s+13\), and we want to write it in the form \((s+c_1)^2 + c_2\). We make the observation that \((s+c_1)^2 = s^2 + 2c_1 s + c_1^2\). So for the coefficient of \(s\) to match, \(2c_1\) must be \(6\). So \(c_1=3\).

Now we have \(s^2 + 6s + 13 = (s+3)^2 + c_2\). If we expand \((s+3)^2 = s^2 + 6s + 9\), then we have

\[ s^2+6s+13 = s^2+6s+9+c_1 \]

we see that to match the constant term, we need \(9+c_2=13\) so \(c_2=4\).

So we now have \(q(s) = (s+3)^2 + 4\). We convert the \(4\) into a square and we have

\[ q(s) = (s+3)^2 + 2^2 \]

Because this is a sum of squares rather than a difference of squares, we cannot do any more factoring (with real numbers).

The process we use will follow Example 1.18:

  1. Find \(c_1\) so that the coefficient of \(s\) is correct.

  2. Use the result to find the value of \(c_2\) so that the constant term is correct.

Example 1.19 (Completing the square when \(q\) has two distinct factors: \(q(s) = s^2-11s+28\))

To complete the square for \(q(s) = s^2-11s+28\), we follow the steps again:

\[\begin{align*} (s+c_1)^2 + c_2 &= s^2-11s+28\\ s^2+2c_1 s+c_1^2 + c_2 &= s^2-11s+28 \end{align*}\]

So \(c_1=-11/2\).

\[\begin{align*} (s-11/2)^2 + c_2 &= s^2-11s+28\\ s^2-11s + \frac{121}{4} + c_2 &= s^2 - 11s+28 \end{align*}\]

Since \(28=112/4\) we have \(\frac{121}{4} + c_2 = \frac{112}{4}\) and \(c_2 = -9/4\).
Thus we have arrived at

\[ q(s) = (s-11/2)^2 - 9/4 \]

Unlike the previous example, this has a difference rather than a sum. We can think of \(9/4\) as \((3/2)^2\). We can use the difference of squares factorization: \(a^2-b^2 = (a-b)(a+b)\).

So

\[\begin{align*} q(s) &= \left( s- \frac{11}{2} - \frac{3}{2}\right) \left( s- \frac{11}{2} + \frac{3}{2} \right)\\ &= (s- 7)(s-4) \end{align*}\]

Example 1.20 (Completing the square when \(q\) is a perfect square: \(q(s) = s^2 - 10s+25\))

To complete the square for \(q(s) = s^2 - 10s + 25\) we follow the same basic steps:

\[\begin{align*} (s+c_1)^2 + c_2 &= s^2 - 10s + 25\\ s^2 + 2c_1 s + c_1^2 + c_2 &= s^2 - 10s + 25 \end{align*}\]

matching coefficients of \(s\), we see \(c_1=-5\). This yields

\[\begin{align*} s^2 - 10s + 25 + c_2 = s^2-10s+25 \end{align*}\]

This leads to the conclusion \(c_2=0\).

So

\[ q(s) = (s-5)^2 \]

Using this process on any quadratic polynomial \(q(s)\) with leading coefficient \(1\) will lead to a factorization of \(q(s)\) (if \(c_2\leq 0\)) or to converting it into the form \((s-a)^2 + \omega^2\).

1.6.3. Finding \(\mathcal{L}^{-1}[p(s)/q(s)](t)\) if \(q(s) = (s-a)(s-b)\)#

Now that we can classify \(q(s)\) based on its factorization, we will use partial fractions to break up \(p(s)/q(s)\) into the sum of two things that are in the Laplace Transform Table. If \(q(s)\) has two real roots, we write:

\[ \frac{p(s)}{q(s)} = \frac{A}{s-a} + \frac{B}{s-b} \]

Then we use Algebraic techniques to find \(A\) and \(B\). We lead by example, and then summarize the steps:

Example 1.21 (Finding \(f(t)\) when \(\tilde{f}(s) = \frac{s-7}{s^2-8s+15}\))

Consider \(\tilde{f}(s) = \frac{s-7}{s^2-8s+15}\). From completing the square (not shown) or other factorization methods we get \(s^2-8s+15 = (s-3)(s-5)\). Expanding \(\tilde{f}(s)\) with partial fractions:

\[\begin{align*} \frac{s-7}{(s-3)(s-5)} &= \frac{A}{s-3} + \frac{B}{s-5}\qquad \text{multiply by $q(s)=(s-3)(s-5)$}\\ \Rightarrow \quad s-7 &= A(s-5) + B(s-3) \\ \end{align*}\]

These equations are true for all values of \(s\). If we choose some convenient values, namely s=3\( and \)s=5\( we can find \)A\( and \)B\( easily because these each make a term on the right hand side become \)0$.

For \(s=3\)

\[ -4 = A(-2) \quad \Rightarrow \quad A = 2 \]

and for \(s=5\)

\[ -2 = B(2) \quad \Rightarrow \quad B =-1 \]

So

\[ \tilde{f}(s) = \frac{2}{s-3} - \frac{1}{s-5} \]

and taking the inverse transform we have

\[\begin{align*} f(t) &= \mathcal{L}^{-1}\left[\frac{2}{s-3} - \frac{1}{s-5}\right](t)\\ &= 2 \mathcal{L}^{-1} \left[\frac{2}{s-3}\right](t) - \mathcal{L}^{-1} \left[\frac{1}{s-5}\right](t)\\ &= 2e^{3t} - e^{5t} \end{align*}\]

The general approach (as above) when \(q(s) = (s-a)(s-b)\) is to

  • Write

\[ \frac{p(s)}{q(s)} = \frac{A}{s-a} + \frac{B}{s-b} \]

  • Multiply through by \(q(s)\) to get

\[p(s) = A(s-b)+B(s-a)\]

  • set \(s=a\) and solve for \(A\) and set \(s=b\) and solve for \(B\).

  • Invert the transform of the resulting sum.

Notice that we leave all of the expressions in Rewritten Denominator form. We never expand a product of polynomials in these steps.

1.6.4. Finding \(\mathcal{L}^{-1}[p(s)/q(s)](t)\) if \(q(s) = (s-a)^2\)#

\[ \frac{p(s)}{q(s)} = \frac{A}{(s-a)^2} + \frac{B}{s-a} \]

Example 1.22 (Finding \(f(t)\) when \(\tilde{f}(s) = \frac{3s+14}{s^2+8s+16}\))

Consider \(\tilde{f}(s) = \frac{3s+14}{s^2+8s+16}\). From completing the square (not shown) or other factorization methods we get \(s^2+8s+16 = (s+4)^2\). Expanding \(\tilde{f}(s)\) with partial fractions:

\[\begin{align*} \frac{3s+14}{(s+4)^2} &= \frac{A}{(s+4)^2} + \frac{B}{s+4}\\ \Rightarrow \quad 3s+14 &= A + B(s+4)\\ \end{align*}\]

These equations are true for all values of \(s\). The value \(s=-4\) is convenient to try because it makes the coefficient of \(B\) become \(0\).

For \(s=-4\):

\[ 2 = A \]

Next we use another value of \(s\). There is no really good second choice like there was for the previous example, but \(s=0\) is often convenient.

So

\[ 14 = A + 4B= 2+4B \Rightarrow 3=B \]

Now

\[ \tilde{f}(s) = \frac{2}{(s+4)^2} + \frac{3}{s+4} \]

and taking the inverse transform we have

\[\begin{align*} f(t) &= \mathcal{L}^{-1}\left[\frac{2}{(s+4)^2} + \frac{3}{s+4}\right](t)\\ &= 2 \mathcal{L}^{-1} \left[\frac{2}{(s+4)^2}\right](t) + 3\mathcal{L}^{-1} \left[\frac{1}{s+4}\right](t)\\ &= 2 e^{-4t} \mathcal{L}^{-1}\left[\frac{1}{s^2}\right](t) + 3 e^{-4t}\\ &= 2te^{-4t} + 3e^{-4t} \end{align*}\]

We used the \(s\)-shift to get the first term in the next to last line and then we used \(t = \mathcal{L}^{-1}[1/s^2](t)\).

So

\[ f(t) = 2te^{-4t} + 3e^{-4t} \]

The general approach (as above) when \(q(s) = (s-a)^2\) is to

  • Write

\[ \frac{p(s)}{q(s)} = \frac{A}{(s-a)^2} + \frac{B}{(s-a)} \]

  • Multiply through by \(q(s)\) to get

\[p(s) = A + B(s-a)\]

  • Set \(s=a\) to make the coefficient of \(B\) zero and find \(A\). Then use a different value of \(s\) to find an equation with \(A\) and \(B\) and solve for \(B\).

  • Invert the transform of the resulting sum (likely using the \(s\)-shift).

Note - in the step to find \(B\), a more efficient approach is to look at the leading coefficient of \(s\). This gives \(B\) directly, but the approach won’t work as well when the denominator is not quadratic.

1.6.5. Finding \(\mathcal{L}^{-1}[p(s)/q(s)](t)\) if \(q(s) = (s-a)^2+\omega^2\)#

\[ \frac{p(s)}{q(s)} = \frac{A(s-a)}{(s-a)^2+\omega^2} + \frac{B\cdot \omega}{(s-a)^2+\omega^2} \]

Example 1.23 (Finding \(f(t)\) when \(\tilde{f}(s) = \frac{4s+3}{s^2+4s+29}\))

Consider \(\tilde{f}(s) = \frac{4s+3}{s^2+4s+29}\). From completing the square (not shown) we get \(s^2+4s+29 = (s+2)^2+25\). Expanding \(\tilde{f}(s)\) with partial fractions:

\[\begin{align*} \frac{4s+3}{(s+2)^2 + 25} &= \frac{A}{(s+2)^2+25} + \frac{B\cdot 5}{(s+2)^2+25}\\ \Rightarrow \quad 4s+3 &= A(s+2) + 5B\\ \end{align*}\]

We see that setting \(s=-2\) is convenient to find \(B\). We get \(-5=5B\) so

\[B= -1\]

To find \(A\), we choose \(s=0\) to get

\[ 3 = 2A + 5B = 2A-5 \Rightarrow 8=2A \]

and so

\[A=4\]

Then

\[ \tilde{f}(s) = \frac{4(s+2)}{(s+2)^2+25} - \frac{5}{(s+2)^2+25} \]

and taking the inverse transform we have

\[\begin{align*} f(t) &= \mathcal{L}^{-1}\left[\frac{4(s+2)-5}{(s+2)^2+25}\right](t)\\ &= 4 \mathcal{L}^{-1} \left[\frac{s+2}{(s+2)^2+25}\right](t) - \mathcal{L}^{-1} \left[\frac{5}{(s+2)^2+25}\right](t)\\ &= 4e^{-2t}\cos(5t) - e^{-2t}\cos(5t) \end{align*}\]

using the \(s\)-shift to get the first term in the next to last line and \(t = \mathcal{L}^{-1}[1/s^2](t)\).

So

\[ f(t) = 4e^{-2t}\cos(5t) - e^{-2t}\cos(5t) \]

The general approach (as above) when \(q(s) = (s-a)^2+\omega^2\) is to

  • Write

\[ \frac{p(s)}{q(s)} = \frac{A(s-a) + B\omega}{(s-a)^2+\omega^2} \]

  • Multiply through by \(q(s)\) to get

\[p(s) = A(s-a) + B\cdot\omega\]

  • Set \(s=a\) to find \(B\). Then use another value of \(s\) to find \(A\).

  • Invert the transform of the resulting sum.

As in the previous example, trying to match the coefficient of \(s\) is more efficient than solving the equation for the second value of \(s\). However, again it will not be as simple when our denominators are no longer quadratic.

1.6.6. Rational Functions with the \(t\)-shift#

Sometimes we encounter functions like \(\tilde{f}(s) = e^{-as}p(s)/q(s)\) where \(p(s)\) and \(q(s)\) are polynomials. We will still approach this using Partial Fractions, but first we need to apply the \(t\)-shift.

Example 1.24 (Finding \(f(t)\) when \(\tilde{f}(s) = \frac{e^{-3s}(3s+14)}{s^2+8s+16}\))

For \(\tilde{f}(s) = \frac{e^{-3s}(3s+14)}{s^2+8s+16}\), when we try to find \(f(t)\), we will first use the \(t\)-shift:

\[\begin{align*} f(t) &= \mathcal{L}^{-1}\left[\frac{e^{-3s}(3s+14)}{s^2+8s+16}\right](t)\\ &= H_3(t)\mathcal{L}^{-1}\left[\frac{3s+14}{s^2+8s+16}\right](t-3) \end{align*}\]

Now, if we can find \(\mathcal{L}^{-1}\left[\frac{3s+14}{s^2+8s+16}\right](t)\), then we just have to substitute \(t-3\) and multiply by \(H_3(t)\) to get the solution to the original problem. This is a rational function, and the techniques we learned above apply.

We can use Partial Fractions to complete this. In fact, in Example 1.22 we saw this example already and we found that

\[ \mathcal{L}^{-1}\left[\frac{3s+14}{s^2+8s+16}\right](t) = 2te^{-4t} + 3e^{-4t} \]

Thus our solution is

\[\begin{align*} f(t) &= H_3(t) \left( 2(t-3)e^{-4(t-3)} + 3 e^{-4(t-3)} \right)\\ &= H_3(t) \left(2t e^{-4(t-3)} - 3e^{-4(t-3)}\right) \end{align*}\]

In general when \(\tilde{f}(s)\) is an exponential times a rational function, we apply the \(t\)-shift first, and then we use Partial Fractions on the remaining part. If \(\tilde{f}(s)\) involves a sum of rational functions times exponentials, we apply this to each term separately.
